**Alphabet** is a high-performance, minimalist programming language where every keyword is a single character. 

---

## üì¶ How to Download Alphabet (Binaries)
Alphabet is automatically built for Windows and Linux via our cloud pipeline. You do not need to install the source code.

### Step-by-Step Download:
1.  Go to the **[Actions](https://github.com/fraol163/alphabet/actions)** tab at the top of this GitHub page.
2.  Click on the most recent workflow run (it should be named **"Build Alphabet Language"** or your last commit message).
3.  Once the page opens, scroll all the way to the bottom to the **"Artifacts"** section.
4.  Download the file for your system:
    -   **`alphabet-windows-latest`**: For Windows (Contains `alphabet.exe`).
    -   **`alphabet-ubuntu-latest`**: For Linux (Contains `alphabet` binary).
5.  Unzip the folder to find your executable.

---

## üõ† OS Setup & Global Installation

### üêß For Linux Users
1.  **Extract:** Unzip the downloaded artifact.
2.  **Permission:** `chmod +x alphabet`
3.  **Global Install:** `sudo mv alphabet /usr/local/bin/alphabet`
4.  **Test:** Type `alphabet -v`

### ü™ü For Windows Users
1.  **Extract:** Unzip the downloaded artifact to find `alphabet.exe`.
2.  **Organize:** Create a folder at `C:\Alphabet` and move the `.exe` there.
3.  **Path Setup:**
    -   Open Start Menu, type **"Environment Variables"**.
    -   Select **Path** under "System Variables" and click **Edit**.
    -   Click **New** and add `C:\Alphabet`.
4.  **Test:** Open a new PowerShell and type `alphabet -v`
