# Alphabet Language — Book/Course Outline

## "Learn Programming with Alphabet"

### Target Audience
- Complete beginners (no programming experience)
- Non-English speakers
- Students aged 12+
- Self-learners

### Book Structure (20 chapters)

#### Part 1: Getting Started (Chapters 1-5)
1. **Introduction to Programming** — What is programming? Why Alphabet?
2. **Your First Program** — Hello World, running code
3. **Variables and Types** — Numbers, strings, booleans
4. **Arithmetic and Operators** — Math, comparison, logic
5. **Input and Output** — z.o(), z.i(), user interaction

#### Part 2: Control Flow (Chapters 6-10)
6. **Making Decisions** — if/else, ternary
7. **Loops** — for, while, for-each
8. **Functions** — Declaration, parameters, return
9. **Error Handling** — try/handle
10. **Pattern Matching** — match/case

#### Part 3: Data Structures (Chapters 11-15)
11. **Lists** — Creation, access, builtins
12. **Maps** — Key-value pairs, JSON
13. **Strings** — Manipulation, formatting
14. **Sets** — Unique collections
15. **Functional Programming** — map, filter, reduce

#### Part 4: Object-Oriented Programming (Chapters 16-18)
16. **Classes** — Fields, methods, constructors
17. **Inheritance** — extends, abstract, interfaces
18. **Encapsulation** — Public, private, static

#### Part 5: Real-World Projects (Chapters 19-20)
19. **File I/O and JSON** — Reading/writing files, JSON processing
20. **Building a Project** — CLI tool, calculator, game
