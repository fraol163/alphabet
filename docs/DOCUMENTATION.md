## Standard Library
- `z.o(x)` : Print
- `z.i()`  : Input
- `z.f(p)` : Read File
- `z.fw(p, data)` : Write File
- `z.sqrt(x)` : Square Root
- `z.abs(x)` : Absolute Value
- `z.pow(a, b)` : Power
- `z.len(x)` : Length (string/list/map)
- `z.type(x)` : Type Name
- `z.sort(list)` : Sort List
- `z.keys(m)` : Map Keys
- `z.values(m)` : Map Values

## REPL Commands
- `help` : Show help
- `clear` : Clear screen (redraws banner)
- `clear history` : Clear command history
- `clear all` : Clear screen + history
- `reset` : Reset state
- `vars` : Show variables
- `keywords` : Show keywords
- `history` : Show last 20 entries
- `!N` : Recall history item N
- `!!` : Repeat last entry
- `#alphabet<en|am|es|fr|de>` : Switch language
