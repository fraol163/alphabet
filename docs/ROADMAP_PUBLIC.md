# Alphabet Language — Public Roadmap

## Current: v2.3.5 (May 2026)

### ✅ Completed
- Core language (19 keywords, 5 languages)
- Bytecode VM with 45 opcodes
- 82 builtins
- OOP (classes, inheritance, interfaces)
- Functional programming (map/filter/reduce, closures)
- 21 stdlib modules
- LSP server for VS Code
- Voice-to-text input
- NL-to-Code converter
- CI/CD + nightly releases
- Embedding API
- BNF grammar spec
