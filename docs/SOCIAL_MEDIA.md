## Reddit Posts

### r/programming
```
Title: I built a multilingual programming language that supports 5 languages

I've been working on Alphabet, a programming language that lets you code in
English, Amharic, Spanish, French, or German. It features:

• 19 keywords (single-letter for experts, full words for beginners)
• Voice input (speak your code!)
• Functional programming (map/filter/reduce, closures)
• 70+ built-in functions
• LSP server for VS Code
• Open source (MIT license)

The language compiles to bytecode and runs on a custom VM built in C++17.

I'd love feedback from the community. What features matter most to you?

GitHub: https://github.com/alphabet-lang/alphabet
```
