# Alphabet Language Error Messages (i18n)

## English (en)
- E001: "Expect expression."
- E002: "Expect ')' after expression."
- E003: "Expect ']' after index."
- E004: "Expect '}}' after block."
- E100: "Cannot import module: {path}"
- E101: "Type mismatch: cannot assign type {src} to variable of type {dst}"
- E200: "Stack overflow"
- E201: "Stack underflow"
- E202: "Undefined variable '{name}'"
- E203: "Instruction limit exceeded ({limit}). Possible infinite loop."
- E204: "Cannot call non-function value"
