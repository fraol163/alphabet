# Lesson 1: Hello World

## What you'll learn
- How to print output
- The basic structure of an Alphabet program

## Concept
Every program needs a way to show output. In Alphabet, `z.o()` prints text to the screen.

Think of it like a speaker — you put text in, and it comes out for everyone to see.

## Syntax
```
z.o("your text here")
```

## Example
```
#alphabet<en>
z.o("Hello, World!")
```
