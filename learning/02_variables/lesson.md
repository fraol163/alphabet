# Lesson 2: Variables

## What you'll learn
- How to store values in variables
- Named types: `int`, `str`, `float`

## Concept
A variable is like a labeled box. You give it a name, put something inside, and retrieve it later.

- `int` = whole numbers (1, 42, -10)
- `str` = text ("hello", "Alice")
- `float` = decimal numbers (3.14, 2.5)

## Syntax
```
int age = 20
str name = "Alice"
float pi = 3.14
```
