# Lesson 3: Basic Math

## What you'll learn
- Arithmetic operators: +, -, *, /, %
- Storing results in variables

## Concept
Programming is great at math. These operators work just like on paper:
- `+` addition
- `-` subtraction
- `*` multiplication
- `/` division
- `%` remainder (modulo)

## Example
```
int result = 10 + 5 * 2
z.o(result)  // prints 20 (multiplication first!)
```
