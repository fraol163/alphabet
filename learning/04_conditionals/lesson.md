# Lesson 4: Conditionals

## What you'll learn
- Making decisions with `i` (if) and `e` (else)
- Comparison operators: ==, !=, <, >, <=, >=

## Concept
A condition is a yes/no question. The program decides what to do based on the answer.

Real life: "If it's raining, take an umbrella. Otherwise, wear sunglasses."

## Syntax
```
i (condition) {
  // runs if true
} e {
  // runs if false
}
```
