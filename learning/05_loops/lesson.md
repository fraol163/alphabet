# Lesson 5: Loops

## What you'll learn
- Repeating code with `l` (loop)
- For-loop syntax with init : condition : increment

## Concept
A loop repeats code until a condition is false.

Real life: "Wash each dish until the sink is empty."

## Syntax
```
l (5 i = 0 : i < 5 : i = i + 1) {
  z.o(i)
}
```
This prints: 0, 1, 2, 3, 4
