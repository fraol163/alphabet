# Lesson 6: Functions

## What you'll learn
- Defining reusable functions with return types
- Parameters and return values

## Concept
A function is a reusable recipe. You write it once, call it many times.

Real life: A coffee recipe. You follow the same steps every morning.

## Syntax
```
int add(int a, int b) {
  r a + b
}
z.o(add(3, 4))  // prints 7
```

`r` = return (sends a value back to the caller)
