# Lesson 7: Lists

## What you'll learn
- Creating and accessing lists
- List operations: append, length, indexing

## Concept
A list is an ordered collection. Like a shopping list — each item has a position (0, 1, 2...).

## Syntax
```
list nums = [10, 20, 30]
z.o(nums[0])     // prints 10 (first item)
z.o(z.len(nums)) // prints 3
z.append(nums, 40)
```
