# Lesson 8: Classes

## What you'll learn
- Defining classes with fields and methods
- Creating objects with `new`

## Concept
A class is a blueprint. An object is an instance of that blueprint.

Real life: "Car" is a class (blueprint). Your red Toyota is an object (instance).

## Syntax
```
c Dog {
  str name = ""
  void bark() {
    z.o(name + " says woof!")
  }
}
Dog d = new Dog()
d.name = "Rex"
d.bark()
```
