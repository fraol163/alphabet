# Lesson 10: Project — Calculator

## What you'll learn
- Combining everything: variables, functions, classes, conditionals

## Concept
Build a simple calculator class that can add, subtract, multiply, and divide.

This project uses:
- Classes (blueprint for calculator)
- Methods (operations)
- Conditionals (choosing operation)
- Return values (getting results)

## Your turn
Complete the Calculator class with the four basic operations.
