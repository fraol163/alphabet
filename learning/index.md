# Alphabet Learning Curriculum

## Level 1 — Beginner (What is programming?)
01. Hello World — Print output with z.o()
02. Variables — Store values with int, str, float
03. Basic Math — Arithmetic operators: +, -, *, /, %

## Level 2 — Fundamentals (Making decisions + repeating)
04. Conditionals — if/else with i/e
05. Loops — Repeat with l (init : cond : incr)

## Level 3 — Organization (Structuring code)
06. Functions — Reusable code with return values
07. Lists — Ordered collections of values

## Level 4 — OOP (Modeling the real world)
08. Classes — Blueprints for objects

## Level 5 — Advanced (Real-world patterns)
09. Error Handling — Try/handle for graceful errors
10. Project: Calculator — Combine everything

## How to use
```
alphabet learn           # list all lessons
alphabet learn 01        # show lesson 01
alphabet learn 01 --run  # run exercise and check
```
